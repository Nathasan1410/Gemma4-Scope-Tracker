# Knowledge Strategy

Swara’s “survival pack” is designed to be usable with or without AI.

## Survival Pack Categories (MVP)

- medical
- fire
- flood
- earthquake
- violence / safety threat
- lost / stranded

## Category Buttons (How They Work)

Category buttons select **relevant knowledge scope** for the current situation. They do not switch between different models.

## Region-Priority Concept (Roadmap / Partial)

Region-aware prioritization changes which SOPs appear first:

- Jakarta / Sumatra: flood-first
- Japan: earthquake-first
- Coastal regions: tsunami-first

## Survival Book

The Survival Book is a primary mode: it must remain usable without AI.

## Local Pack Metadata (Required)

Local packs should include metadata:

- version
- date
- source label (where it came from / who curated it)
