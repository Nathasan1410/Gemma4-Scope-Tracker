# Special Tracks (Technical Strategy)

This repo supports multiple implementation tracks. Only one should be treated as primary.

## Tracks

- **LiteRT (LiteRT-LM)**: primary track for the mobile special track runtime.
- **Unsloth**: optional fine-tuned SLM experiment (P1 story); not an MVP blocker.
- **llama.cpp**: roadmap path for resource-constrained Linux/Raspberry Pi / host-node mode.
- **Cactus**: skip unless real multi-model routing exists (avoid fake “router” claims).
- **Ollama**: non-core; acceptable for desktop testing only.
